# FinanceAppFinalProject
Please follow these steps in order to run my project:
Step 1: After getting all the folders, please open it up it "FinanceAppFinalProject"
Step 2: Open a Command Prompt or Terminal window in the folder containing the published application.
Step 3: run the application using the dotnet run command
Step 4: The application will start and provide a URL (usually http://localhost:[port_number]) that you can access in your web browser.
